# Maru Online - Optimized Visitor Journey Flows

## Executive Summary

Based on analysis of the current website structure and lead generation engine, I've identified 4 primary visitor journey flows optimized for different visitor types and entry points. Each flow is designed to maximize conversion while providing value at every touchpoint.

## Primary Entry Points Analysis

### Current Entry Points:
1. **Homepage** (`/`) - Primary landing page
2. **Assessment Tools** (`/assessments/*`) - Direct tool access
3. **Services Pages** (`/services/*`) - Service-specific landing
4. **Knowledge Base** (`/knowledge/*`) - Content marketing
5. **Website Audit** (`/website-audit`) - Lead magnet
6. **Organic Search** - Various pages via SEO

## Conversion Optimization Recommendations

### 1. Homepage Optimization
**Current Issues:**
- Multiple CTAs competing for attention
- Assessment showcase buried below hero

**Recommendations:**
```
HERO SECTION:
├── Primary CTA: "Get Free Website Audit" (Lead Magnet)
├── Secondary CTA: "Take 2-Min Assessment" (Value First)
└── Tertiary: "How We Work" (Education)

ASSESSMENT SHOWCASE:
├── Move to position #2 (right after hero)
├── Add "Choose Your Starting Point" headline
├── Emphasize "No signup required"
└── Add social proof numbers
```

### 2. Assessment Tool Flow
**Current Issues:**
- No clear progression between tools
- Results gating could be optimized

**Recommendations:**
```
TOOL PROGRESSION:
├── Lead Score Predictor (Entry point)
├── Pipeline Leak Detector (Upsell)
├── Tech Stack Auditor (Cross-sell)
└── Proposal Accelerator (Advanced)

RESULTS OPTIMIZATION:
├── Show partial results immediately
├── Gate detailed PDF report
├── Offer "Complete Analysis" upgrade
└── Include booking CTA in results
```

### 3. Service Page Integration
**Current Issues:**
- Weak connection between assessments and services
- No clear next steps

**Recommendations:**
```
SERVICE PAGES:
├── Add relevant assessment CTA at top
├── Include "Start with Free Audit" buttons
├── Show assessment results examples
└── Add "Book Strategy Call" prominently
```

# Testimonial Images

This directory contains avatar images for testimonials.

## Current Status
Using CSS-generated initials as placeholders in the Testimonials component.

## To Add Real Images
1. Add avatar images here (avatar-1.jpg, avatar-2.jpg, etc.)
2. Recommended size: 200x200px
3. Format: JPG or PNG
4. Optimize images before adding

## Alternative
Consider using a service like:
- UI Avatars (https://ui-avatars.com/)
- DiceBear (https://dicebear.com/)
- Gravatar
